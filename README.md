# autoimporttemplate

A simple framework that using automatic import module

### Requirements

* Python 2.7+

### Architecture

```
|
--- main
    |
    ---main.py
|
---modules
   |
   ...
```

### Usage

```
python main/main.py -m module_name
```

### Example

```
$python main/main.py -m dummy

prepare something...
Hello world
clean something...
```
